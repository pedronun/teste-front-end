# Ninja Som - Feito por Pedro Lucas

## Tecnologias usadas

- ReactJs
- Typescript
- Sass
- React-router-dom
- React-icons
- Axios

Para rodar o projet no localhost execute os seguinte códigos:

Primeiramente clone o repositório para a máquina local e digite no prompt

```sh
cd ninjasom
yarn install
yarn start
```
